# 贡献指南

感谢参与华秋开源硬件项目。

## 提交 Issue

请选择最接近的问题模板，并提供：

- 项目名称和硬件版本
- EDA、编译器、SDK 或工具链版本
- 供电、连接方式和外设环境
- 可重复执行的步骤
- 预期结果与实际结果
- 日志、波形、照片或截图

请勿上传密码、Token、证书、Wi-Fi 凭据、个人信息或无再分发权限的文件。

## 提交 Pull Request

1. 一个 PR 只处理一个明确问题。
2. 更新设计文件时同步更新原理图 PDF、BOM、制造文件和 CHANGELOG。
3. 更新固件时说明编译环境，并确保示例能够从干净环境构建。
4. 新增第三方内容时注明来源和许可证。
5. 不得删除作者署名、版权声明或安全警告。

维护者会优先处理阻断复现、安全风险和明确的文档错误。
