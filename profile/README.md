## 精选方向

- 机器人与边缘 AI：RDK X5、端侧视觉、AI IPC

- 数字电源与功率电子：经过验证并包含安全边界的开源设计

## 仓库发布标准

每个正式仓库应尽可能提供：

- EDA 源文件、原理图 PDF、Gerber/钻孔/坐标文件
- 结构化 BOM、关键器件替代说明和制造注意事项
- 可公开的固件或源码、工具链版本、构建与烧录步骤
- 清晰的 Quick Start、接口定义、供电要求和已知问题
- 分层许可证：硬件、软件、文档、图片和第三方文件分别说明

## 参与贡献

请在对应项目仓库提交 Issue 或 Pull Request。技术问题应包含硬件版本、工具版本、复现步骤、测量条件和相关日志。

涉及未公开漏洞、密钥或安全问题时，请使用仓库 Security 页面私下报告，不要创建公开 Issue。
